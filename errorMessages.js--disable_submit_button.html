disable submit button

Submit button coding


• Code already exists to disable submit button
• Objective then is to add code to add error messages when agree is selected and nothing is filled out and add different “success” messages when agree box is checked and the form is correctly filled out, allowing the form to be submitted as outlined in the Original view of the Register page in html document.

Objectives


1. create function that checks for empty fields in the form and adds error messages when agree box is checked (use ‘change’ event) (This is to prevent user from hitting the register button prematurely)
2. if fields are filled out correctly then ‘success’ message should be added

Coding/debugging process


• Rather than creating new functions to perform the desired tasks I found it more efficient to re-use the existing validation function with some modifications to suit the task.
• To get the correct names for the fields to insert into the error messages I tried looking at the DOM elements to use parentElement.textContent but found the resulting string too complex to be really useful. Instead I took advantage of the data attribute in the html where I could easily get a reference for the desired message
